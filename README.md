
## 소개
안녕하세요 웹 개발자 Berkley 입니다. 저는 이전에 부트캠프와 알고리즘 공부는 Java로 하였지만, 현재 주력으로 하는 스택은 React입니다. 현재 React 라이브러리를 30퍼 이상 사용을 하였으며, 반응형 웹앱 개발이 가능한 수준입니다.
